# Fleqx

**Fleqx**: Normalizing Flows for [JAX](https://github.com/jax-ml/jax) in [distreqx](https://github.com/lockwo/distreqx).

This library provides normalizing flows and utilities using `distreqx` distributions and bijectors.

Current support is minimal and very experimental, using [FlowJAX](https://github.com/danielward27/flowjax) as a backend. However, the goal is to eventually have the same flows as `FlowJAX` implemented natively in `distreqx`.

## Installation
Fleqx can be installed using pip:

``
pip install fleqx
``# fleqx
